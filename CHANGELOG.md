## v0.2 (2026-07-22) — 新角色：尔波

### 新增角色
- 🌊 **尔波**（HP 125 / ATK 13 / 速度 10）
  - **天赋·波纹机制**：无波纹时，每受到3次伤害随机收集一种波纹（共6种）
  - **奥义**：无需任何条件限制，有波纹即可释放，消耗波纹触发对应效果
  - 6种波纹效果：
    | 波纹 | 效果 |
    |------|------|
    | 🌊 尔波纹 | 连续10次攻击（每次等于当前攻击力） |
    | 💧 水波纹 | 吸取敌人10%最大生命值 |
    | 🔥 火波纹 | 敌人进入灼烧，每回合扣5%最大HP（持续至战斗结束） |
    | 🌿 木波纹 | 免疫下次伤害，并反弹10%真伤 |
    | ⚜️ 金波纹 | 攻击力永久+5 |
    | 🪨 土波纹 | 敌人攻击力-20%，持续2回合 |

### 系统更新
- 图鉴·道具页新增波纹系统完整说明
- 地图界面为尔波显示波纹收集进度/当前波纹

## v0.1 (2026-07-22) — 初始版本

### 核心系统

- **Roguelike 回合制爬塔**：12层关卡，每层4个节点（3场普通战斗 + 1场Boss战），共48场战斗
- **14位可选寻宝人**：12生肖角色 + 2位联动角色（soso5王、快餐侠洲歌）
- **回合制战斗系统**：速度决定先手，支持暴击、闪避、吸血、毒素等机制
- **终结技系统**：每位角色拥有独特的终结技，需满足条件才能释放，每层限用一次
- **镇镇之力系统**：5种隐藏能力，满足特定条件后觉醒，每层可用一次
- **奖励选择系统**：战斗胜利后三选一奖励（基础强化/补给/唯一道具）
- **商店系统**：每层通关后进入营地商店，可用寻宝币购买装备和补给
- **背包系统**：记录已获得的唯一道具
- **官方图鉴**：浏览所有寻宝人、怪物、道具和系统说明
