# 12俊俊寻宝记 x 联动扩展版

> 一款 Roguelike 回合制爬塔 RPG 网页游戏。选择你的寻宝人，穿越12层险境，挑战Boss，收集道具，击败最终Boss【俊俊鸡谭】！

---

## 快速开始

直接用浏览器打开 `index.html` 即可游玩，无需任何构建工具或服务器。

---

## 游戏特色

- **14位可选角色**：12生肖 + 2位联动角色（soso5王、快餐侠洲歌），每位拥有独特天赋和终结技
- **Roguelike爬塔**：12层 × 4节点 = 48场战斗，难度逐步递增
- **回合制战斗**：速度决定先手，支持暴击、闪避、吸血、毒素、反伤等机制
- **终结技系统**：每位角色有独特的终结技，满足条件后释放，每层限用一次
- **镇镇之力**：5种隐藏能力，满足特定条件后觉醒，每层可用一次，足以扭转战局
- **奖励选择**：战斗胜利后三选一，可叠加基础属性或获取唯一道具
- **营地商店**：每层通关后购买装备和补给
- **隐藏Boss**：🦆 鸭俊（10%概率遭遇，击败掉落神器）
- **联动内容**：soso5王 + 快餐侠洲歌 专属角色、道具、机制
- **内置图鉴**：浏览所有角色、怪物、道具和系统说明

## 如何游玩

### 1. 选择角色
在选人界面点击任意寻宝人开始游戏。每个角色有不同的属性、天赋和终结技。

### 2. 战斗流程
- 每层有4个节点，前3个为普通战斗，第4个为Boss战
- 速度高的一方先攻击
- 每回合自动进行，可在右上角切换1倍/2倍速
- 满足条件后可使用**终结技**（每层限一次）
- 满足隐藏条件后可使用**镇镇之力**（每层限一次）

### 3. 战后奖励
- 战斗胜利获得寻宝币，三选一奖励
- Boss战额外赠送固定回血奖励
- 击败隐藏Boss鸭俊必掉神器「鸭俊板板」

### 4. 营地商店
- 每层通关后进入营地，终结技和镇镇之力重置
- 可用寻宝币购买装备和补给（三选一）

### 5. 通关目标
击败第12层第4节点的最终Boss【俊俊鸡谭】即可通关！

## 尔波·波纹系统

尔波通过受击收集波纹，奥义释放波纹（无楼层次数限制）。

| 波纹 | 效果 |
|------|------|
| 🌊 尔波纹 | 连续10次攻击（每次等于当前攻击力） |
| 💧 水波纹 | 吸取敌人10%最大生命值 |
| 🔥 火波纹 | 灼烧：每回合扣5%最大HP（持续至战斗结束） |
| 🌿 木波纹 | 免疫下次伤害 + 反弹10%真伤 |
| ⚜️ 金波纹 | 攻击力永久+5 |
| 🪨 土波纹 | 敌人攻击力-20%，持续2回合 |
